A simple and effective Python project that evaluates the strength of a password based on multiple security rules. It helps users understand how strong or weak their password is and suggests improvements.

OVERVIEW OF THE PROJECT:
This project analyzes any user-entered password and checks it against several security criteria such as length, presence of numbers, uppercase/lowercase letters, special characters, etc.
Based on these checks, it returns whether the password is Weak, Moderate, or Strong.

FEATURES:
1.Checks password length
2.Detects uppercase & lowercase letters
3.Detects digits
4.Detects special characters
5.Displays strength level (Weak / Moderate / Strong)
6.Gives suggestions to improve password strength
7.Simple Python CLI program

TECHNOLOGIES AND TOOLS USED:
Python 3.x, Basic Python libraries (no external dependencies), VS Code, Github, Gitbash, Git, Some AI tools for guidance.

STEPS TO INSTALL AND RUN THE PROJECT:
1. Clone the repository: git clone <your-repo-link>
2. Navigate to project folder: cd password-strength-checker
3. Run the Python file: python passwordchecker.py
